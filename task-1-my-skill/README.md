# Task 1: My Daily-Life Skill

**Project Title:** Daily Fitness & Diet Logger
**AI Tools Used:** Claude

**What it does:** It takes messy, quick inputs about my daily activities and meals and turns them into a structured health log. I chose this because it helps me track my physical fitness and diet goals effortlessly.

**Prompts Used:**
*Initial Prompt:* "Use the skill-creator skill to build me a daily fitness logger. Whenever I tell you my steps and what I ate, format it into a clean log with sections for Physical Activity, Diet & Nutrition, and a Daily Summary."

**How I verified it worked:**
I opened a brand-new chat, gave it a normal request without naming the skill ("Today I ran 19k steps and ate 600 gram chicken"), and confirmed it automatically triggered the exact format specified in the SKILL.md.
